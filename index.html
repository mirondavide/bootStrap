0. Come Includere Bootstrap

Ci sono due modi per includere Bootstrap nel tuo progetto:

Metodo 1: Download

Scarica i file di Bootstrap e includili localmente nel tuo progetto:

Scarica Bootstrap 5.3.8 
<!-- Nel <head> - CSS di Bootstrap -->
<link href="css/bootstrap.min.css" rel="stylesheet">

<!-- Prima di </body> - JavaScript di Bootstrap -->
<script src="js/bootstrap.bundle.min.js"></script>
Metodo 2: CDN

Usa i link CDN per includere Bootstrap direttamente da internet (non serve scaricare nulla): 

<!-- Nel <head> - CSS di Bootstrap -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- Prima di </body> - JavaScript di Bootstrap -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>

3. Spaziatura (Margin e Padding)

Formato: {proprieta}{lato}-{dimensione}

Proprieta
  • m = margin
  • p = padding
Lato
  • t = top (alto)
  • b = bottom (basso)
  • s = start (sinistra)
  • e = end (destra)
  • x = sinistra + destra
  • y = alto + basso
  • (vuoto) = tutti i lati
Dimensione
  • 0 = 0
  • 1 = 0.25rem
  • 2 = 0.5rem
  • 3 = 1rem
  • 4 = 1.5rem
  • 5 = 3rem
Esempi
m-3 p-3 (margin e padding su tutti i lati)
mt-2 mb-4 (margin-top 2, margin-bottom 4)
px-5 py-2 (padding asse x 5, asse y 2)

6. Carousel

Il Carousel e uno slideshow per scorrere tra immagini o contenuti.

  • .carousel - Contenitore principale
  • .carousel-inner - Contiene le slide
  • .carousel-item - Ogni singola slide
  • data-bs-ride="carousel" - Attiva l'autoplay
<div id="demoCarousel" class="carousel slide w-25 mx-auto" data-bs-ride="carousel">
    <div class="carousel-inner">
        <div class="carousel-item active">
            <img src="img.jpg" class="d-block w-100" alt="Slide 1">
        </div>
        <div class="carousel-item">
            <img src="img.jpg" class="d-block w-100" alt="Slide 2">
        </div>
        <div class="carousel-item">
            <img src="img.jpg" class="d-block w-100" alt="Slide 3">
        </div>
    </div>
    <button class="carousel-control-prev" type="button" data-bs-target="#demoCarousel" data-bs-slide="prev">
        <span class="carousel-control-prev-icon"></span>
    </button>
    <button class="carousel-control-next" type="button" data-bs-target="#demoCarousel" data-bs-slide="next">
        <span class="carousel-control-next-icon"></span>
    </button>
</div>
